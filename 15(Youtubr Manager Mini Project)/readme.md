# YouTube Manager

A terminal-based YouTube Manager built using Python.

## Features

* List all videos
* Add new videos
* Update existing videos
* Delete videos
* Count total videos
* Store data permanently using JSON

## Concepts Used

* Python Functions
* Lists and Dictionaries
* File Handling
* JSON
* Exception Handling
* CRUD Operations
* Loops and Conditional Statements

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python main.py
